Classes used to build parsers of the XML-encoding to be used with the JAXP optional package.

The classes here can be used to create X3D specific parsers under the Java API for XML Processing (JAXP) optional package.

Property Files

The following property fiels are in use by classes in this package.
org/web3d/x3d/jaxp/containerfields.props
Mapping of node element names to the default containerField attribute value. Used when the XML parsing is not validating and so we don't get the values directly passed to us.
