Common renderer-independent implementation for custom Xj3D extension supporting rigid body physics modelling.

The nodes here provide the basic field handling needed for the nodes, including checking for validity of code structure and ranges. A specific renderer would need to extend these nodes with their own interfaces with the specific rendering API.

Specification

This is a custom extension to Xj3D. It will not work in any other browser. The specification can be found at: http://www.xj3d.org/extensions/rigid_physics.html
