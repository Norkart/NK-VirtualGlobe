An Eclipse View plugin of the Xj3D Browser.

Note that this package requires a set of Version 3.2 Eclipse libraries in your class path to compile and be compatible with Xj3D. These can be located at: eclipse project downloads. The minimum set of Eclipse libraries required for compilation are:

  • org.eclipse.core.commands
  • org.eclipse.core.jobs
  • org.eclipse.equinox.common
  • org.eclipse.equinox.registry
  • org.eclipse.jface
  • org.eclipse.ui.workbench
Additional libraries from the Eclipse package are required to run. The Eclipse RCP-SDK should provide a suitable base execution environment.
