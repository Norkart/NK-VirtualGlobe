The Swing versions of the EAI and SAI browser panels for Xj3D are customizable in appearance. The J3DBrowserFactoryImpl, X3DJ3DBrowserFactoryImpl, VRMLOGLBrowserFactoryImpl, and X3DOGLBrowserFactoryImpl factories attempt to load the properties file xj3d-skins.properties when first instanced. This properties file defines the cursors to use for each navigation mode, the images to use for the buttons in the navigation toolbar, the images for the viewpoint navigation toolbar and the image for the show browser console button.

The browser factories search in various places and use the first copy of xj3d-skins.properties located. Default property values are used if the loaded property file does not specify a value for the property and are NOT substituted in the event of an error.

The default images are located in the src/images directory of the Xj3D codebase.

touchSensor.cursor
Cursor for mouse over enabled touch sensor. Default is images/navigation/CursorTarget.gif
dragSensor.cursor
Cursor for mouse dragging on enabled touch sensor. Default is images/navigation/drag.gif
anchor.cursor
Cursor for mouse over enabled anchor. Default is images/navigation/link.gif
WALK.cursor
Cursor for walk mode. Default is images/navigation/CursorWalk.gif
EXAMINE.cursor
Cursor for examine mode. Default is images/navigation/CursorExamine.gif
PAN.cursor
Cursor for pan/slide mode. Default is images/navigation/CursorPan.gif
FLY.cursor
Cursor for fly mode. Default is images/navigation/CursorFly.gif
TILT.cursor
Cursor for tilt or rotate in place mode. Default is images/navigation/CursorTilt.gif
WALK.button
Button image to select walk mode. Default is images/navigation/ButtonWalk.gif
FLY.button
Button image to select fly mode. Default is images/navigation/ButtonFly.gif
EXAMINE.button
Button image to select examine mode. Default is images/navigation/ButtonExamine.gif
PAN.button
Button image to select pan/slide mode. Default is images/navigation/ButtonPan.gif
TILT.button
Button image to select tilt/rotate in place mode. Default is images/navigation/ButtonTilt.gif
CONSOLE.button
Button image to show browser console window. Default is images/navigation/ButtonConsole.gif
HOME.button
Button image for the viewpoint home button. Default is images/navigation/ButtonHome.gif
PREVIOUSVIEW.button
Button image for the previous viewpoint button. Default is images/navigation/ButtonBack.gif
NEXTVIEW.button
Button image for the next viewpoint button. Default is images/navigation/ButtonForward.gif
