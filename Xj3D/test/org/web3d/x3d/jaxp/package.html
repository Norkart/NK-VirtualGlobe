Classes used to build custom parsers to be used with the JAXP optional package.

The classes here can be used to create X3D specific parsers under the Java API for XML Processing (JAXP) optional package. The custom builders will create a DOM model that is also using the Scene Authoring Interface (SAI)

DOM Building

The document builder factory will parse an X3D document and build a DOM tree from it. The DOM tree will then also have the capability of using the SAI methods to interact with it or operate in mixed mode.

To use the document builder, you need to set the system property javax.xml.parsers.DocumentBuilderFactory to the value org.web3d.x3d.jaxp.DocumentBuilderFactoryImpl. @see SAI doc @see Sun's XML>
